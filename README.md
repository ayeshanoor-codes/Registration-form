# Registration Form  

A simple and responsive **HTML & CSS Registration Form** designed for beginners to practice form fields, input validation structure, and clean UI layout.  
This project includes multiple input types such as text, email, password, date, select dropdown, and textarea.

---

## Features  
- Clean and modern UI  
- Fully responsive layout  
- Hover and focus effects  
- Includes all basic form fields  
- User-friendly structure  

---

## Technologies Used  
- HTML5  
- CSS3  

---

## How to Use  
1. Download or clone the repository.  
2. Open the **index.html** file in your browser.  
3. Edit the form as needed to suit your project requirements.
